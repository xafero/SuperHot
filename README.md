# SuperHot [![NuGet](https://img.shields.io/nuget/v/SuperHot.svg)](https://www.nuget.org/packages/SuperHot/) [![GitHub builds](https://github.com/xafero/SuperHot/actions/workflows/dotnet.yml/badge.svg)](https://github.com/xafero/SuperHot/actions)

<img align="right" width="160px" height="160px" src="logo.png">

This is a Renesas/Hitachi SuperH instruction decoder and disassembler.

## Overview
This project should disassemble the following:

| Hitachi | Example chips                          |
| --------|----------------------------------------|
|  SH-1   | SH7020, SH7021, SH7032, SH7034         |
|  SH-2   | SH7047, SH7049, SH7105, SH7107, SH7109 |
|  SH-2A  | SH7201, SH7206, SH7211, SH7214         |
|  SH-2E  | SH7055, SH7058                         |
|  SH-3   | SH7706, SH7709S                        |
|  SH-4   | SH7091, SH7750                         |
|  SH-4A  | SH7450, SH7451, SH7734, SH7785         |
